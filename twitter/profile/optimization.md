## 4. Profile Picture Recommendations

### Option A: Face (Personal Brand)
**Pros:**
- Builds trust and connection
- People follow people, not logos
- Easier to grow personal brand alongside BetBrain
- More engaging for Twitter algorithm

**Cons:**
- Ties your identity to performance
- Less scalable if you sell the project

**Recommendation:** Use a **high-quality headshot** with:
- Neutral or slight smile (approachable but professional)
- Good lighting (natural or studio)
- Simple background (solid color or blurred)
- Square crop, face centered

### Option B: Logo (Brand Focus)
**Pros:**
- Professional, scalable
- Separates you from the product
- Easier to sell/license later

**Cons:**
- Less personal connection
- Harder to stand out in feeds
- Can feel corporate/impersonal

**Recommendation:** If using a logo:
- Simple brain icon with circuit patterns
- Green (#00FF88) on black background
- Minimalist, recognizable at small sizes

### Size Specs (Both Options)
- **Dimensions:** 400 x 400 pixels (minimum)
- **Display:** Cropped to circle (keep important content centered)
- **Format:** PNG or JPG
- **File Size:** Under 2MB

### Style Guidance
```
DO:
✓ High resolution (400x400 minimum)
✓ Good contrast (visible at thumbnail size)
✓ Simple background
✓ Face/logo centered

DON'T:
✗ Cluttered backgrounds
✗ Text in profile pic (unreadable at small size)
✗ Low resolution / pixelated
✗ Busy patterns or gradients
```

**Final Recommendation:** Start with **Option A (Face)** — Twitter rewards personal brands. You can always switch to a logo later if BetBrain scales.

### Link Tree vs Single Link

**Use a Single Custom Landing Page** (NOT Linktree)

**Why:**
- ✅ Full control over branding
- ✅ No Linktree ads/branding
- ✅ Better for SEO
- ✅ Can add email capture
- ✅ More professional
- ✅ Custom domain builds trust

**Linktree Only If:** You need something up in <1 hour and can't build a landing page

### UTM Tracking Setup

**UTM Parameters for Twitter Bio Link:**
```
Base URL: https://betbrain.ai
UTM Parameters: ?utm_source=twitter&utm_medium=bio&utm_campaign=profile_launch
```

**Full URL:**
```
https://betbrain.ai?utm_source=twitter&utm_medium=bio&utm_campaign=profile_launch
```

**Track These Sources Separately:**
```
Twitter Bio:      ?utm_source=twitter&utm_medium=bio
Pinned Tweet:     ?utm_source=twitter&utm_medium=pinned_tweet
Regular Tweets:   ?utm_source=twitter&utm_medium=tweet
```

### Google Analytics Setup
1. Create GA4 property for betbrain.ai
2. Set up conversion events:
   - `view_tracking_sheet` (when they click to tracking sheet)

3. Create audience segments by UTM source
