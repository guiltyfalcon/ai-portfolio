# BetBrain AI - Canva Setup Instructions

## Overview
Step-by-step guide to creating all BetBrain AI templates in Canva. Optimized for Twitter/X content creation.

## Part 1: Account Setup

### Step 1: Create Canva Account
1. Go to [canva.com](https://canva.com)
2. Sign up for free account (or Canva Pro for premium features)
3. **Recommended:** Canva Pro ($12.99/month) for:
   - Brand Kit (save colors, fonts, logos)
   - Background remover
   - Premium templates
   - Resize tool (adapt designs for multiple platforms)
   - Transparent PNG downloads

### Step 2: Set Up Brand Kit (Canva Pro)
1. Click **Brand** in left sidebar
2. Click **Add New Brand Kit**
3. Name: "BetBrain AI"

#### Add Brand Colors:
| Color Name | Hex Code | Usage |
|------------|----------|-------|
| BetBrain Green | #00FF88 | Accents, wins, highlights |
| BetBrain Black | #000000 | Backgrounds |
| BetBrain White | #FFFFFF | Primary text |
| Dark Gray | #1A1A1A | Cards, sections |
| Light Gray | #CCCCCC | Secondary text |
| Red (Losses) | #FF4444 | Losses, negative numbers |

**How to Add:**
- Click "Add New Color"
- Enter hex code
- Name the color
- Repeat for all 6 colors

#### Add Brand Fonts:
1. Click **Add New Font**
2. Search and add:
   - **Montserrat** (primary - all weights)
   - **Roboto Mono** or **Space Mono** (for odds/numbers)
   - **Poppins** (alternative to Montserrat)

**How to Add:**
- Search font name
- Click to add
- Repeat for all fonts

#### Add Logo:
1. Click **Uploads** tab
2. Upload BetBrain logo (brain icon or custom logo)
3. Drag to Brand Kit
4. Set as primary logo

## Part 2: Template Creation

### Template 1: Bet Slip Graphic (1080 x 1080px)

#### Step-by-Step:

**1. Create Design**
- Click "Create a design" (top right)
- Custom size: 1080 x 1080 px
- Click "Create new design"

**2. Set Background**
- Click background
- Click color square (top left)
- Select BetBrain Black (#000000)

**3. Add Header Bar**
- Press `R` (rectangle tool)
- Draw rectangle: 1080 x 80px (full width, top)
- Color: Gradient from #000000 to #1A1A1A
- **How to add gradient:**
  - Click rectangle
  - Click color square
  - Click gradient icon (2nd tab)
  - Add 2 colors: Black (#000000) and Dark Gray (#1A1A1A)
  - Direction: Left to right

**4. Add Header Text**
- Press `T` (text tool)
- Type: "🧠 BETBRAIN AI"
- Font: Montserrat Bold
- Size: 48px
- Color: White (#FFFFFF)
- Position: Top left, 40px from left edge, centered vertically in header

**5. Add Lightning Icon**
- Click "Elements" tab
- Search: "lightning bolt"
- Select simple lightning icon
- Color: Green (#00FF88)
- Size: 48px
- Position: Top right, 40px from right edge

**6. Add Pick Title Section**
- Press `T`
- Type: "DAILY PICK"
- Font: Montserrat Medium
- Size: 24px
- Color: Green (#00FF88)
- Position: Center, 120px from top

- Press `T`
- Type: "[Team/Player Name]"
- Font: Montserrat Bold
- Size: 56px
- Color: White (#FFFFFF)
- Position: Center, below "DAILY PICK"

- Press `T`
- Type: "[vs Opponent / Event]"
- Font: Montserrat Regular
- Size: 32px
- Color: Light Gray (#CCCCCC)
- Position: Center, below team name

**7. Add Stats Cards (4 boxes)**
- Press `R`
- Draw rectangle: 480 x 150px
- Color: Dark Gray (#1A1A1A)
- Border: 2px, Green (#00FF88)
- Corner radius: 12px (click corner dots)
- Position: Top left quadrant (120px from left, 400px from top)

- Duplicate 3 times (Ctrl/Cmd + D)
- Arrange in 2x2 grid:
  - Card 1: Top left (Odds)
  - Card 2: Top right (Confidence)
  - Card 3: Bottom left (Units)
  - Card 4: Bottom right (AI Analysis)
- Spacing: 40px between cards

**8. Add Card Labels**
- For each card, add label text:
  - Card 1: "ODDS" (20px, Light Gray, Medium)
  - Card 2: "CONFIDENCE" (20px, Light Gray, Medium)
  - Card 3: "UNITS" (20px, Light Gray, Medium)
  - Card 4: "AI ANALYSIS" (20px, Light Gray, Medium)
- Position: Top of each card, 20px padding

**9. Add Card Values**
- Card 1 (Odds): "[+150 / -110]" (40px, Green, Bold, Roboto Mono)
- Card 2 (Confidence): 
  - Add progress bar (rectangle, 200 x 12px, rounded)
  - Fill: Green (#00FF88), 80% width
  - Background: #333333
  - Text: "82%" (32px, White, Bold)
- Card 3 (Units): "[3U]" (40px, White, Bold)
- Card 4 (AI Analysis): "[Analysis text...]" (24px, White, Regular, 2-3 lines)

**10. Add Footer Bar**
- Press `R`
- Draw rectangle: 1080 x 60px (full width, bottom)
- Color: Black (#000000)

**11. Add Footer Text**
- Press `T`
- Type: "🎯 @holikidTV"
- Font: Montserrat Medium
- Size: 22px
- Color: Light Gray (#CCCCCC)
- Position: Bottom left, 40px from left edge

- Press `T`
- Type: "#BetBrain #Betting"
- Font: Montserrat Medium
- Size: 22px
- Color: Light Gray (#CCCCCC)
- Position: Bottom right, 40px from right edge

**12. Save as Template**
- Click "File" → "Save as template"
- Name: "BetBrain - Bet Slip Template"
- Add to folder: "BetBrain Templates"

## Part 4: Export Settings

### For Twitter Posts:

**Format:** PNG (preferred) or JPG
- PNG: Better quality, supports transparency
- JPG: Smaller file size, good for photos

**Quality:** 100% / Maximum
- Click "Download"
- Select file type
- Quality: Maximum

**Color Profile:** sRGB
- Ensures colors display correctly on all devices

**File Size:** Keep under 5MB
- Twitter's maximum file size
- If over 5MB: Reduce quality to 80-90% or resize

**Resolution:** 72 DPI (screen optimized)
- Higher DPI unnecessary for digital use

### Export Checklist:
- [ ] Correct dimensions (1080 x 1080, 1080 x 1350, etc.)
- [ ] PNG format (or JPG if file size issue)
- [ ] Maximum quality
- [ ] sRGB color profile
- [ ] Under 5MB file size
- [ ] Transparent background (if needed for overlay)

## Part 7: Keyboard Shortcuts (Speed Up Workflow)

### General:
- `T` = Add text box
- `R` = Add rectangle
- `C` = Add circle
- `L` = Add line
- `Ctrl/Cmd + D` = Duplicate selected element
- `Ctrl/Cmd + G` = Group selected elements
- `Ctrl/Cmd + Shift + G` = Ungroup
- `Ctrl/Cmd + C` = Copy
- `Ctrl/Cmd + V` = Paste
- `Ctrl/Cmd + Z` = Undo
- `Ctrl/Cmd + Y` = Redo
- `Delete` = Delete selected element
- `Arrow keys` = Nudge selected element (1px)
- `Shift + Arrow keys` = Nudge selected element (10px)

### Text:
- `Ctrl/Cmd + B` = Bold
- `Ctrl/Cmd + I` = Italic
- `Ctrl/Cmd + U` = Underline
- `Ctrl/Cmd + Shift + >` = Increase font size
- `Ctrl/Cmd + Shift + <` = Decrease font size

### Alignment:
- Select multiple elements
- Click "Position" (top toolbar)
- Choose: Left, Center, Right, Top, Middle, Bottom
- Or use: `T` (top), `B` (bottom), `L` (left), `R` (right), `H` (horizontal center), `V` (vertical center)

## Part 8: Troubleshooting

### Issue: Text looks pixelated
**Solution:**
- Export as PNG (not JPG)
- Use maximum quality
- Ensure font size is 20px+ (smaller text can blur)

### Issue: Colors look different on Twitter
**Solution:**
- Use sRGB color profile
- Check on mobile and desktop
- Some color variation is normal (different screens)

### Issue: File size too large
**Solution:**
- Reduce quality to 80-90%
- Use JPG instead of PNG (if no transparency needed)
- Reduce dimensions (but keep aspect ratio)

### Issue: Template won't save
**Solution:**
- Check internet connection
- Refresh page (Canva auto-saves)
- Try "File" → "Save as template" manually

### Issue: Can't find font
**Solution:**
- Search exact font name (e.g., "Montserrat")
- Some fonts require Canva Pro
- Use alternative: Poppins instead of Montserrat
