```

### Element Specifications

| Element | Font | Size | Weight | Color | Position |
|---------|------|------|--------|-------|----------|
| Brand Header | Montserrat | 60px | Bold | #00FF88 | Top center, 80px from top |
| Logo/Icon | - | 120px | - | #00FF88 | Below header, centered |
| Game Matchup | Poppins | 48px | SemiBold | #FFFFFF | Center, 280px from top |
| Teams | Poppins | 42px | Regular | #FFFFFF | Below matchup |
| Bet Type | Roboto Mono | 36px | Bold | #00FF88 | Center, 520px from top |
| Bet Details | Roboto Mono | 32px | Regular | #FFFFFF | Below bet type |
| Confidence | Montserrat | 36px | Bold | #00FF88 | Below details |
| Status Badge | Montserrat | 42px | ExtraBold | #00FF88 | Bottom, 880px from top |
| Hashtag | Poppins | 28px | Regular | #FFFFFF | Bottom, 950px from top |

### Step-by-Step Canva Instructions

1. **Create Design**
   - Click "Create a design" → "Custom size"
   - Enter: 1080 × 1080 px
   - Click "Create new design"

2. **Set Background**
   - Click background
   - Color picker → Enter `#0A0A0A`

3. **Add Brand Header**
   - Text → "Add a heading"
   - Type: "BETBRAIN AI"
   - Font: Montserrat, Bold, 60px
   - Color: `#00FF88`
   - Position: Top center

4. **Add Logo/Icon**
   - Elements → Search "brain icon" or "AI icon"
   - Choose minimalist tech icon
   - Color: `#00FF88`
   - Size: 120px
   - Position: Below header

5. **Add Game Matchup**
   - Text → "Add a subheading"
   - Font: Poppins, SemiBold, 48px
   - Color: `#FFFFFF`
   - Position: Center vertical

6. **Add Bet Details Box**
   - Elements → Shapes → Rectangle
   - Color: Transparent or `#00FF88` at 10% opacity
   - Border: `#00FF88`, 2px
   - Add text inside with Roboto Mono

7. **Add Status Badge**
   - Elements → Shapes → Rounded rectangle
   - Color: `#00FF88`
   - Text: "✅ LOCKED IN"
   - Font: Montserrat, ExtraBold, 42px
   - Color: `#0A0A0A` (reverse)

8. **Add Hashtag**
   - Text → "Add a little bit of body text"
   - Type: "#BetBrainAI"
   - Font: Poppins, 28px
   - Color: `#FFFFFF`

┌──────────────────────────────────────────────────────────┐

│                                                          │

│   │                                                │    │

│   │                                                │    │
│   └────────────────────────────────────────────────┘    │

│   🧵 THREAD                          [Arrow/Icon]        │  ← Footer (40px)
│                                                          │

```

### Element Specifications

| Element | Font | Size | Weight | Color | Position |
|---------|------|------|--------|-------|----------|
| Brand Name | Montserrat | 32px | Bold | #00FF88 | Top left, 40px from top |
| Logo | - | 60px | - | #00FF88 | Top right, 40px from top |
| Main Title | Montserrat | 72px | ExtraBold | #FFFFFF | Center, 200px from top |
| Subtitle | Poppins | 36px | Regular | #FFFFFF | Below title |
| Visual Frame | - | 600×300px | - | #00FF88 border | Center, 350px from top |
| Thread Badge | Montserrat | 40px | Bold | #00FF88 | Bottom left, 580px from top |
| Decorative Icon | - | 48px | - | #00FF88 | Bottom right |

### Step-by-Step Canva Instructions

1. **Create Design**
   - "Create a design" → "Custom size"
   - Enter: 1200 × 675 px

2. **Set Background**
   - Background color: `#0A0A0A`

3. **Add Top Bar**
   - Elements → Line
   - Position: 50px from top
   - Color: `#00FF88`, 1px thickness
   - Length: Full width

4. **Add Brand Elements**
   - Text: "BETBRAIN AI"
   - Font: Montserrat, Bold, 32px, `#00FF88`
   - Position: Top left
   - Add logo icon top right

5. **Add Title**
   - Text → Heading
   - Font: Montserrat, ExtraBold, 72px
   - Color: `#FFFFFF`
   - Center horizontally, 200px from top
   - Max width: 900px (centered)

6. **Add Subtitle**
   - Text → Subheading
   - Font: Poppins, 36px
   - Color: `#FFFFFF` at 80% opacity
   - Position: 40px below title

7. **Add Visual Frame (Optional)**
   - Elements → Rectangle frame
   - Size: 600 × 300px
   - Border: `#00FF88`, 3px
   - Position: Center, below subtitle

8. **Add Thread Badge**
   - Text: "🧵 THREAD"
   - Font: Montserrat, Bold, 40px
   - Color: `#00FF88`
   - Position: Bottom left

9. **Add Decorative Element**
   - Elements → Search "arrow" or "chevron"
   - Color: `#00FF88`
   - Position: Bottom right

## How to Duplicate for Reuse

### Method 1: Canva Templates
1. Create your first design
2. Click "File" → "Make a copy"
3. Rename: "BetBrain - Bet Slip - Template"
4. Keep this as your master template
5. For each new post: Open template → "File" → "Make a copy" → Edit content

### Method 2: Brand Kit (Canva Pro)
1. Go to "Brand Hub" (left sidebar)
2. Click "Add new" → "Brand Kit"
3. Name: "BetBrain AI"
4. Add brand colors: `#00FF88`, `#0A0A0A`, `#FFFFFF`
5. Add brand fonts: Montserrat, Poppins, Roboto Mono
6. Upload logo (if you have one)
7. Now colors/fonts appear in quick-select for all designs

### Method 3: Template Folder
1. Create folder: "BetBrain Templates"
2. Save all 4 master templates here
3. Create subfolder: "Published" for finished graphics
4. Duplicate from master → Edit → Move to Published

## Export Settings

### For Twitter/X
- **Format:** PNG (recommended) or JPG
- **Quality:** 100% (maximum)
- **Size:** Full dimensions (do not resize)
- **Transparency:** Not needed (solid black background)

### Export Steps
1. Click "Share" (top right)
2. Click "Download"
3. File type: **PNG**
4. Quality: **100%**
5. Pages: All (or select specific page)
6. Click "Download"

### File Naming Convention
```
betbrain-betslip-2026-03-05.png
betbrain-thread-ai-picks-explained.png
betbrain-recap-week-10.png
betbrain-lock-lakers-vs-celtics.png
```
