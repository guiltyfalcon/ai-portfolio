## Client Process: Inquiry to Delivery

### Step 1: Initial Contact (0-24h)
1. Client fills out contact form on website
2. Form submission goes to Formspree → forwards to our email
3. We respond within 24 hours with:
   - Confirmation of receipt
   - Clarifying questions (if needed)
   - Recommended package
   - Payment instructions

### Step 2: Discovery & Deposit (24-48h)
1. Client completes detailed brief questionnaire:
   - Business description
   - Target audience
   - Competitor references
   - Design preferences (colors, style)
   - Must-have features
   - Existing brand assets (logo, photos)
2. Client pays 50% deposit
3. We confirm project start date

### Step 3: Design & Development (48-72h)
1. Create initial design mockup/wireframe
2. Build landing page using:
   - HTML5 + Tailwind CSS
   - Responsive design
   - Conversion-optimized layout
3. Deploy to staging URL for client review

### Step 4: Revision Round (72-96h)
1. Client reviews staging site
2. Submits revision requests via email or Telegram
3. We implement changes (unlimited for Pro/Enterprise)
4. Final approval from client

### Step 5: Launch & Final Payment (96h+)
1. Client pays remaining 50%
2. Deploy to production (Vercel or client host)
3. Set up analytics and tracking
4. Hand over all source files
5. 30-day support period begins (Enterprise only)

## Deployment Instructions

### Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Navigate to agency directory
cd ~/git/guiltyfalcon/ai-portfolio/agency

# Deploy
vercel --prod

# Follow prompts to link project
```

### Custom Domain Setup

1. Purchase domain via Cloudflare or Namecheap
2. Add domain in Vercel dashboard
3. Update DNS records as instructed
4. Wait for SSL provisioning (automatic)
